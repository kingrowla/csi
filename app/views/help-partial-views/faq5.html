Update Root Certificate on Server

If the root certificate in your web server is corrupted, or your CIS application server can not connect to secure.collectorsolutions.com you may need to download the root certificate update from Microsoft.

Troubleshooting Tips To Consider Prior to Installing Update
On the server or workstation that you are having connectivity issues, first open up a command prompt, and type the following:

ipconfig /flushdns

This will flush your DNS cache from the workstation or server.

Then open up a browser and go the following web site:

https://secure.collectorsolutions.com/test.htm

If you are able connect, then you DO NOT NEED TO UPDATE YOUR ROOT CERTIFICATE.

If you are not able to reach the site, and receiving a 404 error: unable to reach the web site, then you will need to update the certificate. Click on the link below to download and install the update from Microsoft's website.

 

Quick Overview:

This item updates the list of root certificates on your computer to the list that is accepted by Microsoft as part of the Microsoft Root Certificate Program.

This item updates the list of root certificates on your computer to the list that is accepted by Microsoft as part of the Microsoft Root Certificate Program. Adding additional root certificates to your computer enables you to use Extended Validation (EV) certificates in Internet Explorer 7, a greater range of security enhanced Web browsing, encrypted e-mail, and security enhanced code delivery. After you install this item, you may have to restart your computer. Once you have installed this item, it cannot be removed.
