How to Establish a Secured FTP Connection Using Core FTP LE

AUTOMATED BATCH FILES: If you are using a batch file to automate your FTP connection, please note that the FTP command from Microsoft does not support FTPS or SFTP, port 990, nor it supports encrypted SSL connection. There may be third party software out there. If you would like to continue to automate or schedule your FTP functions, you can purchase the full version of CoreFTP or WS-FTP.

 

Establishing an Secured FTP Connection using CoreFTP.

1. Download and install the free version of CORE FTP LE

Click here to Download Core FTP LE

2. After installing, run Core FTP LE.

3. Click on FILE>CONNECT (see image above)

4. You will see a pop up screen (see below), enter all the information as shown.

5. Also, you should also enter your USERNAME* and PASSWORD* in the designated textbox.

6. The, click on the Connect Button.

 

Use the following connections configuration:

 

Note: Under Connection drop down list, choose AUTH SSL or AUTH TLS, and make sure the port is 990.
Port 21 is no longer available.

How do you know if your connection is secure using CoreFTP?



At the bottom right of the application screen you should see a yellow lock. This tells you that your connection is secured and encrypted.
